Asynchronous Process Call Injection

Executing shellcode within a process.

This entry aims to explain what is an Asynchronous Process Call, how does it work and how it can be exploited to execute shellcode.

What is an APC? +

An Asynchronous Procedure Call (APC) is a function that executes asynchronously in the context of a particular thread. On simpler terms, it's a queued code stored within a specific part of a process that runs when such part is free of any task.
